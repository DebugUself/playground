# JWHbk找工作周刊发布手册

~ JW是JobWeekyly的简称

```
找工作过程记录繁杂
记录文档种类也多
但持续记录2个月后
却并无拿到一份面试机会

大妈周会提醒
需要用各种方法尝试
避免死磕

于是受怼周刊启发
编辑发布这本JW杂志
希望达成以下3个目标
1. 形成记录的固定节奏
2. 形成条理清楚的作品
3. 缓解不安紧张情绪
```

### 栏目设置
~ JW周刊分 **常设栏目** 和 **非常设栏目**

- **常设栏目**
    + 定场诗 Poem ~ 由本周关键词、情感组合的艺术品
    + 节点 Deadlines ~ 简报本周重大事件
    + 成果 Achivements ~ 记录本周具体成果
- **非常设栏目** 
    + 故事 Stories ~ 详细报道本周大事件
    + 观点 Voices ~ 有关工作的启发、思考
