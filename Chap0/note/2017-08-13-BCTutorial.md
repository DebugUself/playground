# Python 基本概念教程

## Overview

- 核心数据类型:
    - 数字
    - 文本, UTF-8/中文 文本
    - 列表,嵌套列表,列表推导式 
    - 简单的 for in 循环
    - 字典,嵌套字典
- 常见工程行为:
    - 函式, 完成一个函式,可以拼接两行文本为新文件保存到硬盘
    - 模块, 将函式发布为一个模块,可以在另外一个脚本中调用
    - 文件, 调用此模块,将两个指定 .txt 文件的内容,逐行交错合并为一个新文件
    - 网络, 改进此模块,可以将两个指定网页的内容, 逐行交错合并为一个新文件
